# 考试

## 突击备考是一种技能

没人会计划着靠突击过日子，但大多数人总会有那么一两门课，最后是靠突击救回来的。

有时候是中途掉队了，有时候是几门考试撞在一起，时间根本不够分。到了这种时候，考试周拼的往往不是谁最努力，而是谁更早知道自己该先救哪门、该放哪门。

## 平时怎么避免期末全炸

真到考前最后几天才开始搜资料，通常已经晚了。平时至少做三件事：

第一，把每门课的资料源留好：课件、作业、往年题、重点总结，别到考前再满群问。  
第二，尽早知道这门课到底考什么。老师口头随便提过的一句“这部分不考”，往往比你自己瞎猜半天有用。  
第三，别把每一门课都拖到“以后再看”。哪怕你这学期主打的是苟过去，也要尽量避免所有课一起积烂账。

## 考前三天能做什么

如果你只剩三天，按这个顺序来：

第一步，找一个上课认真的同学，请他吃顿饭，让他花半小时给你讲这门课大概在讲什么、重点在哪、考试考什么类型的题。半小时的口述比你自己翻三天课本高效得多。

第二步，拿到往年考试题。如果有答案就对着答案看，没答案就看题型和出题范围。大部分老师的考试风格是稳定的，连续几年的试卷能告诉你七八成的信息。

第三步，翻课本和课件，只看跟考试范围相关的部分，跳过所有你用不到的内容。把关键公式和概念抄到一张纸上，考前反复看这张纸。

第四步，如果还有时间，做几道练习题。不用全做，做有代表性的几道就行。

## 多门考试撞车时怎么排优先级

不要按“我最讨厌哪门”排序，也不要按“群里谁最慌”排序。按分数和风险排。

先救最危险的那门：挂科风险高、学分高、后续影响大的，优先。  
再救提分快的那门：花几个小时就能明显拉分的，优先。  
最后才是情绪上最折磨你的那门：你最烦它，不代表它最值得你投时间。

考试周最大的错觉就是“每门都还来得及”。其实不是。很多时候真正有效的做法，就是承认你不可能把每门都救到最好，然后把资源投到最该投的地方。

## 不同类型的课，准备方法不一样

闭卷课，重点是题型和范围。你要知道它怎么考，不是从第一页重新感动自己一遍。  
开卷课，重点不是把书带全，而是提前做好索引，知道哪类东西在哪。开卷但不会翻，照样死。  
报告课和论文课，重点是 rubric、示例和老师偏好。别自己感动式输出，先搞清楚老师到底按什么标准给分。

## 考场上

不会做的题不要死磕。先把会做的全部拿到手，再回头看不会的。很多人考试分数低不是因为不会，是因为在前面的难题上耗了太多时间，后面简单的题没时间写。

写满。尤其是大题，哪怕你不确定，把相关的公式和思路写上去，多少能拿点步骤分。空着一分都没有。

## 哪些努力看起来很努力，其实很废

有几种考前行为，做的时候特别容易产生“我很拼”的幻觉，实际上更多只是让自己没那么内疚。

从第一页开始重看课本。  
重抄一遍笔记，抄得很工整。  
在群里和别人互相表演焦虑。  
专挑最难的题做，然后把简单题放着。  
熬夜熬到人都不清醒了，还以为自己在拼。

这些事的共同点是：做起来很有“我在努力”的感觉，但对提分帮助未必大。

考试周要的是分，不是努力感。
