# 考试

## 突击备考是一种技能

先说清楚：我们不鼓励你靠突击过日子。长期稳定地学才是正路。但现实是，总会有那么一两门课你跟不上进度，或者学期末几门考试撞在一起，时间不够分。这种时候你需要知道怎么在有限的时间里最大化你的分数。

## 考前三天能做什么

如果你只剩三天，按这个顺序来：

第一步，找一个上课认真的同学，请他吃顿饭，让他花半小时给你讲这门课大概在讲什么、重点在哪、考试考什么类型的题。半小时的口述比你自己翻三天课本高效得多。

第二步，拿到往年考试题。如果有答案就对着答案看，没答案就看题型和出题范围。大部分老师的考试风格是稳定的，连续几年的试卷能告诉你七八成的信息。

第三步，翻课本和课件，只看跟考试范围相关的部分，跳过所有你用不到的内容。把关键公式和概念抄到一张纸上，考前反复看这张纸。

第四步，如果还有时间，做几道练习题。不用全做，做有代表性的几道就行。

## 考场上

不会做的题不要死磕。先把会做的全部拿到手，再回头看不会的。很多人考试分数低不是因为不会，是因为在前面的难题上耗了太多时间，后面简单的题没时间写。

写满。尤其是大题，哪怕你不确定，把相关的公式和思路写上去，多少能拿点步骤分。空着一分都没有。
