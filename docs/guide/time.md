# 你的时间值多少钱

算一笔账。

本科四年，扣掉寒暑假和睡觉吃饭的时间，你真正可支配的清醒时间大概一万小时出头。这一万小时是你手里唯一的资本。学费已经交了，沉没成本不用管，唯一的变量是你把这些时间花在哪。

假设你毕业后第一年的月薪是一万块（在深圳不算高），你一年工作大概两千小时，时薪五十。那你现在每个小时的"机会成本"至少是五十块。实际上比这更高，因为你现在的每一小时不光产出当下的价值，还在塑造你三年后、五年后值多少钱。

现在回头看看你昨天的时间花在哪了。

躺床上刷了两小时手机？一百块。去参加了一个什么也学不到的社团例会？又是两小时。帮谁跑了个腿、排了个队？一小时。算下来一天可能扔掉了两三百块的时间，而且你甚至不会注意到。

这不是说你每一分钟都得在"产出"。人需要休息，需要发呆，需要和朋友吃饭扯淡。但有一种浪费是无意识的：你不是在休息，也不是在做有用的事，你只是在消磨。不知道干什么，所以刷手机；不好意思拒绝，所以去开会；别人都在做，所以你也做。

有一个办法可以帮你判断一件事值不值得做：如果有人给你五十块钱，让你在这个时间去做这件事，你干不干？如果答案是不干，那你现在就是在免费做一件连你自己都不愿意花五十块做的事。

大学四年最贵的不是学费。是时间。而且花完了不会有人退给你。
