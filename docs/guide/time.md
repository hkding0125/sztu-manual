# 你的时间值多少钱

大学里最容易产生的一种错觉是：时间很多。

课表没有排满，一周看起来还有很多空格，于是两小时短视频、一次没意义的例会、一个顺手答应的帮忙，都像没什么。等学期过完回头看，真正把人拖垮的，往往不是某个大错，而是时间一直在漏，你自己还没察觉。

如果你对“时间管理”这四个字天然反感，可以先用一个很粗暴的算法把自己敲醒。假设你毕业后第一年月薪一万，折下来时薪大概 60 块。这个数不精确，但够用了。以后再碰到一件事，你先问自己一句：如果现在有人给我 60 块，让我花一小时去做这件事，我愿不愿意？

答案要是不愿意，你至少该知道，自己正在付什么代价。

刷一小时手机当然不是犯罪，跟朋友吃饭、散步、发呆也不叫浪费。真正贵的，是那种你既没在休息，也没在做有用的事，只是因为不知道干什么，所以顺手把时间填掉的状态。

## 但别把自己活成时间管理机器

最容易走偏的一点是：你看完这套算法之后，开始对每一分钟都斤斤计较，最后把自己活成一个只会算投入产出比的机器。

那也没必要。人不是工时表。你去散步、谈恋爱、跟朋友扯淡，这些都很正常。真正的问题是，你有没有在主动用时间，还是一直被别人的安排、群消息和自己的惯性拖着走。

时间管理最重要的不是塞满，而是知道什么该留白，什么不能再拖。

## 一个够用的周检法

如果你现在还不会系统规划，那先别搞什么复杂方法。每周只做这三件事就行：

- 回头看一眼：这周时间主要花在哪三件事上
- 问自己一句：这三件事里，哪一件最不值
- 下周砍掉一个低价值固定动作

比如少去一次没有意义的例会，少答应一次其实可以拒绝的顺手帮忙，少掉进一个明明只是用来逃避的短视频坑。

你不需要一周就把人生调好，但你至少该知道，自己这周最不值的一小时花在哪了。这个问题想清楚了，下周通常就会好一点。
