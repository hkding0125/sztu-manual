# 选课

## 你选的不是课，是老师

同一门课，不同老师教出来完全是两个东西。一个给分大方、讲课清楚的老师能让你用最少的精力拿到好成绩；一个点名严格、考试变态的老师能让你一学期的时间全砸在这一门课上。

选课之前先打听老师。问学长学姐、看课程评价、翻往年的考试难度和给分分布。这些信息值得你花时间去搜集，因为它决定了你这个学期的时间分配。

## 成绩权重

每门课的成绩构成不一样：有的平时分占 40%，有的期末考占 80%。这个比例直接决定了你的策略。

平时分比重大的课，出勤和作业就是硬通货，期末考砸了也有兜底。期末占比大的课，平时可以省点力，但最后几天得拼命。选课的时候看清楚成绩构成，别等到期末才发现自己平时分已经没了。

## 关于点名

有的老师每节课点名，有的一学期都不管。这个信息对你的时间分配影响很大。一门不点名的课，如果你自学效率更高，那去不去上课就是你自己的判断了。一门每次点名的课，缺勤直接扣分，你就必须到场。

提前搞清楚哪些课必须去、哪些课可以灵活安排，然后把省出来的时间花在更值得的地方。
