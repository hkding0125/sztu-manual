# 思维篇

## 绩点

每年都有新生觉得只要 GPA 够高就万事大吉。高中确实是这样，高考分数就是一切。但大学不是。

你去考研面试，导师问你做过什么项目。你去投简历，HR 问你解决过什么问题。你去申请留学，对方想看你的研究经历。这些场合下一张成绩单什么都说明不了。

绩点当然要保，这是门槛，过不了就没有后面的事。但你花在从 3.5 刷到 3.8 上的时间，如果拿去做一个项目或者找一段实习，回报可能大得多。每个专业第一名只有一个人，为了争这个位置把所有时间砸进去，不划算。

## 信息差

坪山在深圳东边，偏。16 号线技术大学站直达校门口，14 号线沙田站也在附近，到福田中心区大概 48 分钟。加上走路等车换乘，每次进城实际要一个多小时。

这个距离最大的副作用是让人懒得出门。深圳最好的科技公司、行业活动、实习机会集中在南山和福田，但如果你只看校内群消息，你根本不知道外面在发生什么。竞赛报名、公司宣讲、行业meetup，这些信息在公众号上、在论坛上、在各种微信群里。学校不会转发给你。

深技大是新学校，这一点反而可以利用。很多外部竞赛和项目对新学校有好奇心，你拿学校名头去申请，通过率可能比你以为的高。但前提是你知道有这些东西存在。

## 方向

"你对什么感兴趣？"这个问题让很多人答不上来。答不上来很正常，但到大三还答不上来就有点晚了。

方向不是想出来的，是试出来的。去一个实验室待几个月，你会知道自己喜不喜欢那个东西。学院有三个研究中心，大一大二试错成本最低，多去不同的地方看看。

有人说"我参加个项目看看自己喜欢什么"。想法可以理解，但大部分本科生参与的项目是老师安排好的具体活，你干的是执行，不是探索。一个项目做完你可能知道了一些具体技能，但未必就能因此发现自己的方向。别对单个项目期望太高，当成了解一个领域的窗口就行。
