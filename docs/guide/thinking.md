# 思维篇

## 绩点

很多新生刚进来的时候，会把 GPA 当成大学里的总进度条。这个反应太正常了，毕竟高中就是这么玩的。问题是大学不是。

你去考研面试，导师问你做过什么项目。你去投简历，HR 问你解决过什么问题。你去申请留学，对方想看你的研究经历。这些场合下一张成绩单什么都说明不了。

绩点当然要保，这是门槛，过不了就没有后面的事。更实用的看法不是“GPA 重不重要”，而是“你现在是不是还处在必须先救 GPA 的阶段”。

一个经验上的停手线可以这样看：

- **3.0 以下**：先救命。很多奖学金、交流、申请和老师筛人，连第一眼都过不了。
- **3.0 - 3.3**：还不安全，别太早拿“我靠项目逆袭”安慰自己。你现在最缺的不是故事，是门票。
- **3.4 - 3.6**：对多数路径已经进了能打的区间。这时候再为了 0.1 把所有时间砸进去，要看你是不是明确在冲奖学金、强申请或某些特别吃分的项目。
- **3.7 往上**：如果你不是在冲最看重成绩的那条路，边际收益通常已经开始下降。

这些不是教务处红线，是经验线。真正想说的是：GPA 不能摆烂，但也别把它误当成大学四年的最终产品。

你花在从 3.5 刷到 3.8 上的时间，如果拿去做一个项目或者找一段实习，回报可能大得多。每个专业第一名只有一个人，为了争这个位置把所有时间砸进去，不划算。

## 信息差

深技大最大的信息差，不是你知不知道学校远，而是很多事没人提前告诉你。

学校太新，上面没几届人。你想打听点什么，比如哪个老师怎么样、考研怎么准备、实习去哪找，很多时候没有现成的学长学姐会主动把答案塞给你。成熟大学里那种一届一届传下来的信息链条，这里还没长全。

所以你得自己去找。主动加学长学姐微信，去学院各种群里潜水，看看前几届的人现在在干什么、走了什么路。这些信息比你自己闷头想有用得多。

校内通知渠道能告诉你的东西其实很有限。竞赛报名、公司宣讲、行业活动，很多信息都散在公众号、论坛和各种微信群里。你不主动去找，它们对你来说就跟不存在差不多。

## 方向

“你对什么感兴趣？”这个问题让很多人答不上来。答不上来很正常，但如果一直只靠想，通常也想不出来。

方向这东西，多半不是想出来的，是试出来的。去一个实验室待几个月，你会知道自己到底喜不喜欢那个东西。学院有三个研究中心，大一大二试错成本最低，多去不同的地方看看。

有人说“我参加个项目看看自己喜欢什么”。这个想法可以理解，但大部分本科生一开始进项目，做的都是老师分下来的具体活，你更多是在执行，不是在探索。一个项目做完，你可能会学到一些技能，但未必就能突然找到人生方向。别对单个项目期望太高，把它当成了解一个领域的窗口就行。
