# 思维篇：别在坪山把自己读成了“技校生”

## 1. 警惕“对标德国”的迷魂汤
学校天天说应用技术大学，说对标工业4.0。听听就好，别真以为进了校门你就自动变成工程师了。
- **真相**：实验室的设备确实很贵，但如果你不主动去蹭、去磨，那些设备跟你家里的电视机没区别。
- **策略**：早点看清所谓的“工程实践”到底是在拧螺丝，还是在解决真正的工程逻辑。

## 2. 关于“信息差”的残酷现实
深技大在坪山，物理上被“隔离”了，但你的脑子不能被隔离。
- **别只盯着校内**：深圳最牛逼的资源在南山，在光明，在那些创业公司里。
- **学会白嫖**：利用学校新开的名头，去申请各种外部竞赛、实习和资源。反正学校要名声，你要的是履历。

## 3. 拒绝“高中式”勤奋
很多同学来深技大后，还是每天准时上课、记笔记、考高分，然后就觉得万事大吉。
- **醒醒吧**：在这里，一个能落地的 Demo 远比一张 95 分的卷子有说服力。
- **不要为了 GPA 放弃折腾**：绩点要保，但别为了那 0.1 的差值把自己变成做题机器。

## 4. 德语是你的工具，不是你的主人
如果你在中德班，别整天纠结那些语法。德语是为了让你能直接看懂德国的工程文档，直接和外教吹逼用的。
- 考不过 DSH？那确实惨。但如果你考过了却不敢开口，那这几年德语就白学了。
