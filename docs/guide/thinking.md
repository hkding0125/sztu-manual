# 思维篇：从高中生到独立技术人

## 大学是什么？
在深技大，你可能会听到“我们要对标德国应用技术大学”。这不仅意味着你要学好手艺，更意味着你要学会像工程师一样思考：**发现问题、解决问题、迭代方案。**

## 摆脱被动学习
- **不要等老师喂饭**：官方的培养方案只是底线，不是上限。
- **拥抱开源与社区**：学会使用 GitHub, Stack Overflow, 和各种技术社区。
- **信息差是第一生产力**：学会如何检索信息，而不是在群里当“伸手党”。

## 独立人格
大学是你第一次真正拥有对自己时间的绝对掌控权。
- 拒绝盲目内卷：想清楚你想要的是绩点、还是真正的技术、还是未来的就业竞争力。
- 学会拒绝：拒绝无意义的社团活动，拒绝浪费时间的社交。
