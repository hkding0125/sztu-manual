# 思维篇

## 大学不是高中的延续

很多同学带着高中的惯性进入大学：每天按时上课、认真记笔记、期末考高分，然后觉得自己做得不错。这套做法在高中确实有效——高考就是一场考试，考试成绩就是一切。但大学不是这样运作的。

大学四年结束之后，等待你的不是另一场标准化考试，而是一个需要你拿出真实能力来证明自己的世界。不管你选择考研、留学还是就业，招你的人想看到的都不是"这个人很会考试"，而是"这个人做过什么、能做什么"。一个完成了的项目、一篇参与过的论文、一段有内容可讲的实习经历，在那个时候的说服力远超一个高出零点几的绩点。

这不是说绩点不重要。GPA 是几乎所有筛选环节的门槛指标，该保的分必须保。但问题在于，很多人把全部精力都押在了绩点上，四年下来除了一张成绩单什么都没有。这才是真正危险的地方。

我们的建议是：把绩点当成需要维护的基本面，然后把省下来的时间和精力投入到那些更难被替代的事情上——做项目、进实验室、去实习、参加竞赛。大学四年其实很短，有些事情只有在这个阶段才方便做，过了就没有了。

## 信息差是真实存在的

坪山在地理上确实离深圳的核心区有一段距离。16 号线技术大学站可以把你送到地铁网络里，14 号线沙田站到岗厦北大概 48 分钟——不算太远，但也绝对不近。

但比物理距离更值得警惕的是信息上的隔离。深圳最密集的科技资源、实习机会和行业活动集中在南山和光明，如果你只盯着校内的通知渠道，你能接触到的信息是非常有限的。公众号、行业论坛、技术社群、各种竞赛和活动的官网——这些才是你应该主动去关注的信息源。

另一方面，深技大作为新学校，"新"本身有时候反而是一种资源。很多外部项目和竞赛对新学校有好奇心和开放度，申请的时候不妨大胆试试。学校需要学生去拿成绩来建立声誉，你需要机会来充实履历，这件事其实是双赢的。

## 尽早开始想方向

我们不是说你大一就必须确定一辈子要做什么——那不现实，也没必要。但到大三的时候，你应该对自己想往哪个方向深入有一个大致的判断。

未来技术学院有三个研究中心：未来能源与环境、先进制造与未来工业、创意工业设计。跨学科的机会不少，但也正因为方向多，容易出现"什么都接触了一点、什么都没有深入"的情况。大一大二是试错成本最低的阶段——多去不同的实验室看看，跟不同的老师聊聊他们在做什么，哪些组在招人。这些信息不会自己跑到你面前来，你得自己去找。

方向这个东西，想是想不出来的，得在实际接触中才能慢慢清晰。所以比起坐在宿舍里焦虑，不如先动起来。
