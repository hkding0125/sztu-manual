# 序言

## 为什么会有这本手册

深圳技术大学是一所非常年轻的学校，未来技术学院更是新中之新。这种"新"带来了很多好处——新校区、新设备、新的培养模式——但也意味着一件事：几乎没有什么沉淀下来的经验可以参考。

在一个成熟的大学里，很多重要的信息是通过"学长学姐口口相传"这个非正式渠道传递的：哪个老师的课值得选、实验室怎么找、考研和留学分别要提前多久准备、校园生活有哪些坑要避开。这些东西不会写在招生简章里，也不会出现在学院官网上，但它们实实在在地影响着你的大学四年。

深技大还没有足够厚的校友文化来承载这些信息。于是我们决定把它写下来。

这本手册不是官方出品，也不代表学院的立场。它记录的是在这个学院真实读过书的人的经验和观察。我们尽力确保信息准确——拿不准的东西不会往上写，写错了的地方欢迎来 [GitHub](https://github.com/hkding0125/sztu-manual) 纠正。

## 怎么用这本手册

你不需要从头读到尾。侧边栏有目录，挑你当下最关心的章节看就行。

我们写这些内容的时候，尽量遵循一个原则：少说"你应该怎么想"，多说"你可以怎么做"。如果某一条建议让你觉得有用，拿去用；如果你有不同的看法或者更好的经验，欢迎补充进来。

这本手册的目标很简单：让后来的人少走一些弯路。如果它做到了，那就够了。
