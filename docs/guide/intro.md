# 序言：既然被“流放”到坪山，总得带点东西回去

## 这不是官方手册
如果你想看那些“对标德国”、“国际化一流”的漂亮话，请关掉这个网页去搜官网。

这里记录的是官网永远不会告诉你，但会让你在深技大四年过得不那么狼狈的真相。我们不负责赞美，只负责解构。

## 为什么要写这个？
深技大太新了。新到没有足够厚的校友文化，新到很多信息差甚至能决定你未来的走向。我们不想看你踩那些学长学姐踩烂了的坑。

## 我们的立场
1. **去他妈的内卷**：如果卷是为了更好的未来，那叫奋斗；如果卷是为了在一个没意义的项目里比谁熬夜更晚，那叫自残。
2. **独立人格**：不要被“应用型大学”这个词给框死了。学校把你当高级技工培养，但你得把自己当成一个人。
3. **真实是最高准则**：食堂难吃就是难吃，老师水就是水。
