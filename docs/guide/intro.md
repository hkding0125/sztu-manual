# 序言

深圳技术大学是一所非常年轻的学校，未来技术学院更是新中之新。这种"新"带来了很多好处——新的设备、新的培养模式、新的可能性——但也意味着一个问题：几乎没有什么沉淀下来的经验可以参考。

在一个成熟的大学里，大量重要的信息是通过非正式渠道传递的。哪个老师的组值得去、考研和留学分别要提前多久准备、校园生活有哪些坑要避开——这些东西不会写在招生简章里，也不会出现在学院官网上，但它们实实在在地影响着你的大学四年。深技大还没有足够厚的校友文化来承载这些信息。于是我们决定把它写下来。

这本手册不是官方出品，不代表学院的立场。它记录的是在这个学院真实待过的人的经验和观察。我们尽力确保信息准确——拿不准的东西不往上写，写错了的地方欢迎来 [GitHub](https://github.com/hkding0125/sztu-manual) 纠正。

你不需要从头读到尾。侧边栏有目录，挑你当下最关心的章节看就行。如果某一条建议对你有用，拿去用；如果你有不同的经验，欢迎补充进来。这本手册的目标很简单：让后来的人少走一些弯路。
