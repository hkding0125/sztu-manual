# 序言

你被录取到了深圳技术大学未来技术学院。恭喜。

但接下来的四年怎么过，没有人会教你。学校给你排好课表、安排好宿舍，然后基本上就不管了。你要自己搞清楚该学什么、该跟谁混、该往哪个方向走。搞清楚了的人，毕业的时候手里有东西；没搞清楚的人，四年过完只剩一张毕业证。

这本手册就是想帮你搞清楚。

深技大太新了。2018年才建校，未来技术学院更新。新到很多信息没有沉淀下来，新到你想找个学长问点事情，发现上面没几届人。成熟大学里那套"学长学姐口口相传"的信息系统，这里还没有。哪个老师好、哪个方向有前途、实验室怎么进、毕业了去哪——这些问题你得自己摸索，或者看这本手册。

我们不是官方，不代表学院，写的东西也不一定全对。但至少是真话。觉得哪里写错了，来 [GitHub](https://github.com/hkding0125/sztu-manual) 改。
