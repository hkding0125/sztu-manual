# 序言：关于这本手册

## 这不是官方出品

深圳技术大学未来技术学院有自己的官网，上面有完整的学院介绍、招生信息和新闻动态。如果你在找那些内容，请直接去[学院官网](https://futuretechnologyschool.sztu.edu.cn/)。

这本手册想做的事情不一样：记录在这个学院读书的真实体验——那些官网上不会写、但会切实影响你大学几年的东西。

## 为什么要写这个

深技大是一所很年轻的学校，未来技术学院更是新中之新。很多有用的经验和信息还没有来得及沉淀成"传统"。

学长学姐踩过的坑，如果没人记下来，下一届大概率还会再踩。与其让每个人都自己摸索，不如把这些经验攒在一个地方。

## 几个基本原则

1. **只写确认过的信息**：拿不准的东西不会往上写。如果发现有不准确的地方，欢迎来 GitHub 提 Issue。
2. **给具体建议**：尽量说"你可以怎么做"，少说"你应该怎么想"。
3. **保持友善**：吐槽可以有，但对事不对人。

## 怎么用

侧边栏有目录，挑你当下最需要的章节看就行，不需要从头读到尾。

---

> 如果这本手册能让后来的人少走几步弯路，那就够了。
