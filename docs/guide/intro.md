# 序言

你被录取到了深圳技术大学未来技术学院。恭喜。

但接下来的四年怎么过，没有人会教你。学校给你排好课表、安排好宿舍，然后基本上就不管了。你要自己搞清楚该学什么、该跟谁混、该往哪个方向走。搞清楚了的人，毕业的时候手里有东西；没搞清楚的人，四年过完只剩一张毕业证。

这本手册就是想帮你搞清楚。

深技大太新了。2018 年才建校，未来技术学院也很年轻。新到很多信息还没沉淀下来，新到你想找个学长问点事情，发现上面没几届人。成熟大学里那套“学长学姐口口相传”的信息系统，这里还没有。哪个老师好、哪个方向有前途、实验室怎么进、毕业了去哪，这些问题你得自己摸索，或者看这本手册。

## 这本手册不是给你背的

这不是一本从第一页顺着读到最后一页的教材，也不是学校宣传册的反面版本。

它更像一张路况图。你现在卡在哪，就先看哪一章：

- 刚入学，先看序言、时间、思维，先把脑子摆正
- 这学期快炸了，就看选课、考试、学术篇
- 想找老师、想进组，就看实验室篇
- 已经开始想以后去哪，就看出路篇
- 觉得坪山把人关傻了，就看生活篇

你不用一次全想明白，但你得比昨天更清楚一点自己在做什么。

## 我们写这本的立场

我们不是官方，不代表学院，写的东西也不一定全对。但至少有三件事会尽量做到。

第一，说真话，不拿漂亮口号糊弄你。  
第二，尽量把判断依据讲出来，不只丢结论。  
第三，承认这本手册有视角，有偏见，有适用范围。

如果你看到哪一句觉得“这明显只适合某一类人”，那大概率你看得没错。真正有用的经验，本来就不可能对所有人同时成立。
