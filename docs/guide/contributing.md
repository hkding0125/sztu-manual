# 参与贡献

这是一个开源项目，靠大家一起维护。

## 你可以做什么

- **纠错**：发现信息过时了或者写错了，帮忙更新一下。
- **补充内容**：你在某个方面有经验想分享（比如某个竞赛的参赛攻略、某个方向的选课建议），欢迎写成一篇新章节。
- **改进网站**：如果你懂前端，也欢迎帮忙优化样式和功能。

## 怎么提交

1. Fork [本项目仓库](https://github.com/hkding0125/sztu-manual)。
2. 在 `docs/guide/` 目录下修改或新增 Markdown 文件。
3. 提交 Pull Request，简单说明你改了什么。
4. 审核后会合并到主分支。

## 写作建议

- **信息要准确**：写之前确认一下，拿不准的标注出来。
- **给具体建议**：与其说"多参加课外活动"，不如说"大二上学期可以关注 XX 竞赛，报名截止一般在 X 月"。
- **保持友善**：可以吐槽，但对事不对人。
