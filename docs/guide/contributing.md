# 如何贡献内容

这是一个属于所有深技大人的开源项目。我们非常欢迎你的加入！

## 你可以贡献什么？
1. **内容纠错**：发现某个链接失效了，或者某个食堂关门了。
2. **新增篇章**：你想分享自己在某方面的独特见解（如：考公指南、某专业选课建议）。
3. **技术支持**：如果你擅长前端开发，可以帮我们优化 VitePress 的样式。

## 如何提交？
1. **Fork** 本项目仓库。
2. 在 `docs/guide/` 目录下修改或增加 Markdown 文件。
3. 提交 **Pull Request**。
4. 我们会在审核后将其合并到主分支。

## 文风要求
- **真实**：拒绝官话。
- **干货**：多分享具体操作，少谈宏大叙事。
- **客观**：可以有立场，但不要人身攻击。
