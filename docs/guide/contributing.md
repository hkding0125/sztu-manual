# 参与贡献

这本手册是一个开源项目，你在这里读到的内容都是曾经或正在深技大读书的人写下的。它能持续有用的前提是：有人不断地更新和补充。

如果你发现某条信息已经过时了——比如食堂关了一个窗口、地铁站改了名字、某个项目的选拔条件变了——帮忙更新一下，这对后来的读者是一个很大的帮助。

如果你在某个方面有值得分享的经验，比如某个竞赛的参赛攻略、某个方向的选课建议、实习求职中踩过的坑，欢迎写成一篇新的章节提交上来。

如果你懂前端开发，也欢迎帮我们优化网站本身的体验。

## 怎么提交

1. Fork [本项目仓库](https://github.com/hkding0125/sztu-manual)。
2. 在 `docs/guide/` 目录下修改或新增 Markdown 文件。
3. 提交 Pull Request，简单说明你改了什么、为什么改。
4. 审核后会合并到主分支。

## 关于写作

我们对内容有一个基本要求：信息要准确。拿不准的东西标注出来，不要当成确定的事实来写。

在风格上，我们希望这本手册读起来像是一个认真的学长在跟你聊天——不是发通知，也不是写论文。给具体的建议，把"为什么"讲清楚，保持友善，对事不对人。
