# 学术篇：如何在全英教学和PBL中活下来

## 1. 全英文教学：别被英语卡了脖子
未来技术学院的核心专业课几乎全是全英。
- **真相**：很多时候老师讲得口干舌燥，下面听得一脸懵逼。
- **生存策略**：
    - **预习是救命的**：提前把术语查了，别指望在课堂上现学。
    - **善用 AI 工具**：录音转文字、实时翻译只是辅助，真正的核心是你要建立英文的工程思维。

## 2. 项目制教学 (PBL) 的坑
这里不考卷子，考 Demo。
- **关于组队**：PBL 的成败 80% 取决于队友。远离那些“划水怪”，也别让自己成为那个只会做 PPT 的人。
- **不要为了做项目而做项目**：院士们想看的是你解决问题的逻辑，而不是你买了一个成品模块拼凑出来的玩具。

## 3. 二次选拔：后悔药有吗？
很多人是为了“精英学院”的名号冲进来的。
- 如果你发现自己并不适合高强度的跨学科折腾，尽早考虑转出。
- 这里是给那些“眼里有光、脑子里有代码、手里有泥”的人准备的。
