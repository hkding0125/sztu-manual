# 学术篇：在技术大学如何“降维打击”

## 培养方案解读
（深技大的培养方案通常比传统大学更重实践）
- **核心课**：哪些是你的看家本领？
- **选修课**：哪些能真正拓宽你的视野？

## 关于 GPA (绩点)
- **真相**：在技术型大学，绩点不是万能的，但没有绩点是万万不能的（尤其是保研、考研、出国）。
- **策略**：保持中上游的 GPA，同时拿出 40% 的精力去折腾项目。

## 德语学习：是机遇还是枷锁？
- 如果你在中德合作项目，德语是你连接欧洲工业文明的桥梁。
- 别把它当成一门单纯的考试，试着去读原版的工程手册。

## 实验室与项目
- 尽早联系导师，进入实验室。
- 参加“大创”或各种学科竞赛，不是为了拿奖，是为了体验完整的开发全流程。
