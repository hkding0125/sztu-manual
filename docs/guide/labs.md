# 实验室篇

## 为什么要进实验室

在上一章我们提过，光有绩点是不够的。不管你以后选择考研、留学、走 X-Talent 还是就业，"你做过什么"这个问题都会被反复追问。而实验室是本科阶段最现实的答案来源之一：在这里你可以参与真实的课题、接触前沿的设备、产出论文或项目成果。这些东西在你毕业时的价值，远超多考几分。

但进实验室这件事也有它的坑。盲目地"随便找个组先进去再说"，结果可能是你花了大半年时间做一些机械性的工作，既没学到东西，也没有产出可言。所以在进组之前，想清楚几个问题是有必要的。

## 关于找组

我们见过最常见的误区是冲着"最厉害的老师"去。名气大的老师当然有他的优势——学术资源多、论文平台高——但如果这个老师日理万机没有时间亲自带你，你进组之后实际面对的可能是一个高年级学生，而这个学生给你安排的大概率是组里最边缘的工作。做几个月下来，你发现自己只是在帮别人跑数据或者洗瓶子，谈不上什么成长。

相比之下，一个愿意花时间跟你讨论问题、方向跟你兴趣对口的年轻老师，哪怕名气没那么响，对你的帮助可能大得多。选组这件事，"合适"比"厉害"重要。

具体怎么找？最直接的办法是跟不同的老师聊。Office hour、课后、发邮件都行。你想了解的无非几件事：这个老师在做什么方向、组里现在是什么状态、对本科生有什么期望。这些信息不会自己跑到你面前来。很多组允许你先旁听几次组会——我们建议你在正式加入之前这样做，感受一下氛围和节奏再决定。

学院的研究方向以材料为主，这是一个客观事实。如果你对材料有兴趣，资源不会少。但如果你的兴趣在其他方向，你需要更主动地去寻找——比如用你的编程能力去切入一个材料课题中的计算或数据分析环节。这样做的好处是双重的：老师得到了一个能解决他问题的人，你得到了一个锻炼自己技能的平台。

## 进组之后

刚进组最重要的事情不是表现自己，而是搞清楚情况。组里的规矩是什么、谁负责什么、工作节奏和汇报制度是怎样的——这些基本面先摸清楚，后面的路会顺很多。

有一件事我们想特别强调：实验记录要认真写。这不是形式主义。做研究有一个非常经典的痛苦场景——你三个月前做过一个实验，现在需要复现那个结果或者引用那组数据，但你翻遍笔记本发现当时记得稀稀拉拉，关键参数对不上。认真记录不是在应付老师，是在帮三个月后的自己。

操作设备遇到不确定的地方，先问再动手。学院的很多仪器价值不菲，坦诚地说一句"我不确定这步该怎么操作"远好过自己猜着来然后把东西搞坏。万一真出了问题，第一时间报告。试图隐瞒只会让事情更糟。

最后，如果你进组一段时间后发现方向跟自己的预期不符，不要拖着不说。跟老师坦诚地谈，看能不能调整或者换组。方向不匹配不是你的错，但浪费双方的时间是可以避免的。

## 关于设备

学院的实验设备在同类高校里算一线水准。这些仪器你现在可以免费用，毕业以后想用就得按小时付费了——而且价格不便宜。不管你最终走不走学术路线，"独立完成某某表征"、"熟练操作某某设备"这些经历写在简历上都是加分项。有机会就多接触，这是你在这个学院里能拿到的、最不容易被替代的资源之一。
