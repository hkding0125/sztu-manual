# 实验室篇

## 为什么要进

本科阶段最常被问到的一句话，其实就是：“你做过什么？”实验室是回答这句话最现实的地方之一。

但进组这件事，最怕的不是没进，而是稀里糊涂进了一个不合适的组。花半年帮人跑数据、磨样品、做最边缘的杂活，最后既没学到东西也没留下像样产出，这种事一点都不少。

## 找组

先说清楚立场：下面这部分提醒，更偏向那些**不打算把未来完全押在材料实验上的同学**。如果你本来就想继续走材料、实验、器件这条路，那判断标准会不一样，后面单独说。

最常见的错误是冲着最有名的老师去。名气大的老师学术资源多，但如果他太忙没时间带你，你实际面对的很可能只是一个高年级学生，干的是组里最边缘的活。

找一个方向对口、愿意花时间跟你聊的老师，哪怕名气不大，对你帮助往往更实际。

怎么找？直接跟老师聊。第一次找老师会尴尬很正常，但这一步躲不过。Office hour、课后、发邮件都行。问清楚他在做什么方向、组里什么状态、对本科生什么要求。很多组可以先旁听组会，正式加入之前先去感受一下，比你在外面瞎猜有用。

未技以材料为主。对材料感兴趣的，资源够用；不感兴趣的，自己找别的方向和别的老师，别等着谁来替你分配。

你第一次去聊老师，至少要问明白四件事：本科生进去之后主要干什么，谁在带你，组里最近在推进什么课题，你有没有机会留下像样产出。问清这些，比你回去猜老师“看起来人好不好”有用得多。

### 如果你本来就想做材料

那你要看的不是“材料值不值得做”这种空问题，而是这个老师的方向有没有持续产出、组里本科生有没有成长空间、你进去之后是能真正理解问题，还是只会重复性地磨样品、跑仪器。

材料不是不能做，问题是很多人不是想清楚了才做，而是环境把他默认推了进去。你如果本来就喜欢实验、器件、表征，那就别被网上那种一刀切劝退带着跑，认真看老师、平台、课题质量比什么都重要。

### 如果你不想把未来押在材料上

那就别被“未来技术”这个名字骗得太浪漫。很多资源确实在材料和实验平台上，你可以借，但别把自己最后读成了一个只会操作昂贵仪器的人。

更现实的做法是：利用未技现成的设备、项目和老师资源，同时把自己真正要长出来的能力放在自动化、编程、建模、系统设计或者产品判断上。你可以借这个环境起步，但别让环境替你把路选完。

## 进组之后

先搞清楚组里的规矩，别急着表现。谁管什么、怎么汇报、设备怎么预约，这些弄明白了后面省事。

实验记录认真写。三个月后你需要复现某个实验或者引用某组数据，翻笔记本发现什么都没记，那个痛苦是真实的。

设备不确定怎么操作就问。学院仪器很贵，说“我不确定”比把东西弄坏好。出了问题立刻报告，别藏着。

还有个很现实的提醒：进组两三个月后，你最好能回答“我现在到底学到了什么”。如果这个问题你一直答不上来，只剩下机械干活，那就该重新判断这个组还值不值得待。

方向不对就跟老师说。拖着对谁都不好。

## 设备

学院实验设备在同类高校里算好的。现在免费用，毕业以后按小时收费。有机会多接触。

一个实际问题：别自己一个人硬冲去约器材。很多仪器不是你会不会用的问题，而是有没有老师或高年级同学替你背书。本科生单独去碰壁很正常，让你导师带一下，或者先通过组里的师兄师姐走流程，效率会高很多。
