# 实验室篇

## 先了解大背景

前面在学术篇已经提过，学院的研究方向目前以材料学科为主。这意味着如果你想进实验室，你大概率会接触到材料相关的课题。对材料有兴趣的同学自然如鱼得水，但如果你的兴趣在其他方向，也不必因此就放弃进组的念头——只是需要更主动一些去找到适合自己的位置。

比如，如果你会编程，完全可以在材料课题中承担数据处理、模拟计算或者自动化分析的工作。这样你既拿到了实验室的资源和可能的论文机会，也在锻炼自己真正想发展的技能。在一个以材料为主的学院里，能"带来新工具"的人往往是很受欢迎的。

## 怎么找老师和方向

我们最重要的一条建议是：不要等学院统一安排，主动去找。

具体怎么找？最直接的方式是跟不同的老师聊。Office hour、课后、发邮件都行。你想了解的无非是几件事：这个老师在做什么方向、组里现在缺什么样的人、对本科生的期望是什么、日常的工作节奏和强度大概是怎样的。这些问题你不问，是没有人会主动告诉你的。

选组的时候有一个常见的误区，就是盲目追求"最厉害的老师"。名气大的老师当然有他的优势，但如果这个老师非常忙、没有时间亲自带你，你进去之后可能就是被高年级学生安排做一些边缘性的工作，学到的东西很有限。相比之下，一个愿意花时间带你、方向跟你兴趣对口的老师，哪怕名气没那么大，对你的成长可能更有帮助。

很多组允许你先来旁听几次组会。我们建议你在正式加入之前这样做——感受一下组里的氛围、讨论的深度、成员之间的关系，再做决定。

## 进组之后

刚进组的前几周，最重要的事情是观察而不是表现。搞清楚组里的规矩：谁负责什么、仪器怎么预约、实验记录用什么格式、组会怎么轮转。这些基本的运转规则弄明白了，你后面的日子会顺畅很多。

说到实验记录——认真写。这不是形式主义。科研里有一个非常经典的痛苦场景：你三个月前做过一个实验，现在需要复现或者引用那个数据，但你翻遍笔记本发现当时记得稀稀拉拉，关键的参数和步骤根本对不上。认真记录是在帮未来的自己。

如果在操作设备的时候遇到不确定的地方，先问再动手。学院的实验设备很多都价值不菲，坦诚地说"我不确定这一步该怎么操作"远比自己猜着来然后把东西弄坏好得多。万一真的出了问题，第一时间报告。试图隐瞒只会让事情变得更糟。

最后一点：如果你在组里待了一段时间后发现方向跟自己的预期不符，尽早跟老师坦诚地谈。这不是什么丢人的事情——方向不匹配是很正常的。拖着不说，对你和对老师都是在浪费时间。

## 好好利用硬件条件

学院的实验设备条件在同类高校中算得上一线水准。这些东西你现在可以免费使用，毕业以后想用就得按小时付费了。不管你最终走不走学术路线，"熟练操作某某设备"、"独立完成某某表征"这些经历写在简历上都是实打实的加分项。有机会就多接触，别浪费了。
