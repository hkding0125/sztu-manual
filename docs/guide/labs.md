# 实验室篇：怎么找组、怎么生存

## 1. 先了解大背景：材料为主

学院目前的研究方向**以材料学科为主**。三个研究中心（能源与环境、先进制造、创意设计）听起来覆盖面广，但实际的师资和课题分布里，材料占了大头。

这意味着：
- 如果你对材料感兴趣，资源和机会不会少。
- 如果你的兴趣偏向其他方向（比如软件、自动化、AI），需要更主动地去找适合自己的课题和老师。不是没有，但需要你自己去挖掘。

## 2. 怎么找老师和方向

不要等学院统一分配，主动去找。

- **多跟不同老师聊**：office hour、课后、发邮件都行。了解他们在做什么方向、组里现在缺什么样的人、对本科生有什么期望。
- **选合适的，不一定要选最有名的**：一个愿意花时间带你、方向跟你兴趣对口的老师，比一个名气大但没空管你的老师实际得多。
- **先去旁听组会**：很多组允许你先来旁听几次，感受一下氛围和节奏，再决定要不要正式加入。

## 3. 进组之后

- **先观察再动手**：刚进组的前几周，搞清楚组里的规矩、谁负责什么、工作节奏是什么样的。
- **实验记录认真写**：不是形式主义。三个月后翻回去看，你会庆幸自己当时记得详细。
- **遇到问题主动沟通**：设备操作不确定的先问，别自己猜着来。东西坏了坦诚报告，比事后被发现好得多。
- **方向不对就早说**：在组里待了一阵发现跟自己预期不一样，这很正常。跟老师坦诚聊一下，看能不能调整或换组。拖着不说对谁都不好。

## 4. 跨学科是一个优势

如果你有编程能力或者其他专业背景，试试把这些技能带进实验室。

比如用代码做数据处理、用算法做模拟优化——这样你既拿到了实验室的资源和论文机会，也在锻炼自己想发展的技能。在一个以材料为主的学院里，能"带来新工具"的人往往很受欢迎。

## 5. 设备值得好好利用

学院的实验设备条件不错。这些仪器你现在免费用，毕业以后想用就得花钱了。

不管最终走不走学术路线，"熟练操作某某设备"、"独立完成某某表征"这些经历写在简历上都是加分项。有机会就多接触。
