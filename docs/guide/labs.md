# 实验室篇

## 为什么要进

不管你以后考研、留学、找工作，别人都会问你"做过什么"。实验室是本科阶段回答这个问题最现实的地方。

但别随便找个组就进。进了一个不合适的组，花半年帮人跑数据，既没学到东西也没产出，白耗时间。

## 找组

最常见的错误是冲着最有名的老师去。名气大的老师学术资源多，但如果他太忙没时间带你，你实际上面对的是一个高年级学生，干的是组里最边缘的活。

找一个方向对口、愿意花时间跟你聊的老师，哪怕名气不大，对你帮助更实际。

怎么找？直接跟老师聊。Office hour、课后、发邮件都行。问清楚他在做什么方向、组里什么状态、对本科生什么要求。很多组可以先旁听组会，建议加入之前先去感受一下。

学院以材料为主。对材料感兴趣的，资源够用。不感兴趣的，自己找其他方向的老师，别等分配。

## 进组之后

先搞清楚组里的规矩，别急着表现。谁管什么、怎么汇报、设备怎么预约，这些弄明白了后面省事。

实验记录认真写。三个月后你需要复现某个实验或者引用某组数据，翻笔记本发现什么都没记，那个痛苦是真实的。

设备不确定怎么操作就问。学院仪器很贵，说"我不确定"比把东西弄坏好。出了问题立刻报告，别藏着。

方向不对就跟老师说。拖着对谁都不好。

## 设备

学院实验设备在同类高校里算好的。现在免费用，毕业以后按小时收费。有机会多接触。

一个实际问题：约器材的时候别说自己是本科生，不然大概率不给你用。让你导师跟你一起去，或者让他帮你打个招呼，有人背书才行。
