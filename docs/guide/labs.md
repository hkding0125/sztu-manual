# 实验室与项目：如何避免在“材料坑”里躺平

## 1. 看清现实：这其实是个“巨大的材料学院”
别被那三个高大上的中心名字（能源、制造、设计）给忽悠了。
- **真相**：从底层的师资力量和研究方向来看，这里目前本质上就是一个**“巨大的材料学院”**。如果你对纯材料不感兴趣，你会觉得这里的很多资源对你来说是错位的。
- **预警**：如果你是奔着纯纯的互联网、纯纯的纯软开发来的，你可能会感到很失落。

## 2. 生存法则：主动“套路”老师
既然现有的体系可能不符合你的兴趣，你该怎么办？
- **不要等安排**：学校的“项目制”有时候是按部就班的流程。如果你傻傻地等着学院给你分配导师、分配项目，你大概率会被分去实验室磨样品或者测数据。
- **主动出击（套磁）**：带着你的兴趣去“找”老师。哪怕这个老师的研究方向不是你的首选，但只要你能证明你的想法有价值，**试着让老师进入“你的体系”**。
- **变客为主**：找那些对新技术、新交叉领域感兴趣的老师，用你的方案去打动他们。让老师觉得是在支持一个有潜力的项目，而不是在带一个完成任务的学生。

## 3. 关于资源的建议
- **利用材料背景做跨学科**：如果你真的不喜欢材料，试着把你的兴趣（比如自动化、工业设计、算法）结合到材料应用里。
- **白嫖实验室空间**：不管你做什么，未来学院的实验室条件是顶级的。就算你做的事情和材料无关，也要学会利用这里的场地和设备。
