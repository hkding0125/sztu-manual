# 深造篇：毕业以后去哪儿

## 1. 越早想越好

这个问题不要拖到大四再考虑。大致有这几条路：

- **考研**：走的人最多，竞争也在逐年加剧。
- **X-Talent Program**：学院自己的「4+2+X」培养项目，不用考研也能读硕读博——详见[学术篇](/guide/academic#x-talent-program)。
- **留学**：路径很多，但差别很大——下面会详细说。
- **就业**：深圳的科技公司密度很高，机会不少。
- **考公/考编**：完全正当的选择，但准备周期长，尽早规划。

## 2. 考研

深技大是双非，这个客观事实不需要回避。但双非不等于没有机会。

- 很多导师选人看的是你做过什么、能不能上手干活。带着在学院参与的项目经历和成果去联系导师，比空手套白狼强很多。
- 选学校和导师的时候，"方向匹配"比"学校排名"更实际。找那些研究方向跟你对口、课题组正好缺人的老师，成功率往往更高。
- 英语能力在复试中是一个实实在在的加分项。

## 3. 留学

这里有一个需要特别注意的地方：**区分授课型硕士（MSc）和研究型项目（MPhil / PhD）**。

往届深造去向包括南洋理工大学、悉尼大学、香港理工大学等。但要清楚的是，大部分海外授课型硕士（MSc / MA）是**自费**的，学费加上生活费是一笔不小的开支。

**个人建议**：如果经济条件有限，优先关注**带奖学金的项目**，比如：

- **MPhil**（研究型硕士）：很多学校提供奖学金或助研津贴
- **PhD**（博士）：大部分正规博士项目有全额资助，包括学费减免和生活津贴
- **CSC 奖学金**（国家留学基金委）：可以资助海外读博或联合培养

授课型硕士当然也有它的价值（时间短、转方向灵活），但在做决定之前，把费用算清楚很重要。

其他建议：

- **推荐信**：跟有国际背景的老师保持好关系。一封有分量的推荐信在申请中的权重比很多人以为的要大。
- **GPA 不能塌**：海外院校第一轮筛选基本就看绩点，项目经历再丰富也救不了太低的 GPA。
- **语言考试早点考**：雅思、托福、GRE 这些东西，大二大三搞定最好。拖到大四会跟毕设、申请撞在一起。

## 4. 就业

往届毕业生被腾讯、比亚迪、安永等企业录用过。深圳对应届生总体是友好的，但前提是你提前做了准备。

- **实习尽早开始**：大二暑假就可以试着投。第一份实习不需要多厉害，重要的是开始积累经验。
- **简历写项目和成果**：你做了什么、解决了什么问题——这些比课程列表和 GPA 数字更能吸引 HR 的注意力。
- **应届生身份只有一次**：校招通道、落户政策、专项补贴——这些红利过期就没了，别浪费。

## 5. 其他路径

- **X-Talent Program**：学院的「4+2+X」项目，大三选拔，不用考研，学校资助 15-18 万元，最终拿境外硕士或博士学位。如果你有读研读博的打算，这是一个值得从大一就开始留意的选项。
- **保研 / 推免**：如果绩点和综合排名够，这是成本最低的深造方式。提前了解名额和条件。
- **Gap year**：不是每个人都需要本科一毕业就立刻进入下一阶段。花一年时间工作、实习或准备申请，不丢人。

---

> 选哪条路没有标准答案。但有一条通用的建议：尽早想、尽早准备、别到最后一刻才开始行动。
