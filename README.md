# 深技大生存手册 (SZTU Survival Guide)

本项目受《上海交通大学生存手册》启发，旨在为深圳技术大学的同学提供一份非官方、真实、且具有传承意义的校园生活与学术指南。

## 目录结构 (规划中)

- **[序言](./docs/guide/intro.md)**：既然被“流放”到坪山，总得带点东西回去。
- **[思维篇](./docs/guide/thinking.md)**：别在坪山把自己读成了“技校生”。
- **[学术篇](./docs/guide/academic.md)**：选课、实验室、德语学习与工程实践。
- **[生活篇](./docs/guide/life.md)**：坪山生存指南（吃喝玩乐与交通）。
- **[未来篇](./docs/guide/career.md)**：就业、考研与深造。
